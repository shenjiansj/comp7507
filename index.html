what we used

In this project, we used three different tools to handle the data and build some interactive data visualizations

Python

Python has been used for handle the symptom data, by using some machine learning algorithms we can get the cluster data.

R

R has been used for handle the text by sentiment analysis, there is a pre-defined NLP method can help to handle text data.

Tableau

Tableau has been used for data visulization. We plan to provide some interactive visulization dashboards for end users to understand vaccination.
